# 02_Marketing_Team (마케팅팀)
## Agent Configuration

---

## 🎯 Mission
원소스 멀티유즈(OSMU) 전략으로 하나의 콘텐츠를 다양한 채널에 최적화하여 마케팅 효율 극대화

---

## 👥 Team Structure

### 1. Content Director (콘텐츠 총괄)
- **역할:** 원본 콘텐츠 분석 및 채널별 변환 전략 수립
- **책임:**
  - 핵심 메시지 추출
  - 채널별 톤앤매너 설정
  - 콘텐츠 캘린더 관리

### 2. Visual Director (비주얼 총괄)
- **역할:** 시각적 요소 기획 및 디자인 가이드 작성
- **책임:**
  - 이미지/썸네일 컨셉 기획
  - 브랜드 일관성 유지
  - AI 이미지 생성 프롬프트 작성

### 3. Performance Director (퍼포먼스 총괄)
- **역할:** 채널별 최적화 및 성과 예측
- **책임:**
  - SEO 키워드 전략
  - 해시태그 최적화
  - CTA(Call-to-Action) 설계

### 4. QA Validator (품질 검증)
- **역할:** 브랜드 가이드라인 준수 및 품질 승인

---

## 🔄 Workflow Process

```
[PHASE 1: 원본 분석]
├── Input 폴더에서 소스 콘텐츠 수집
│   ├── 유튜브 링크 → 스크립트 추출
│   ├── 블로그 글 → 핵심 요약
│   └── 주제/키워드 → 리서치 수행
├── 핵심 메시지 3-5개 도출
└── 타겟 오디언스 정의

[PHASE 2: 채널별 변환]
├── 블로그 (SEO 최적화 버전)
├── 인스타그램 (카드뉴스 기획)
├── 뉴스레터 (이메일 마케팅)
├── 트위터/X (스레드 버전)
└── 링크드인 (B2B 톤)

[PHASE 3: 검증 및 최적화]
├── QA: 브랜드 톤 일관성 체크
├── SEO 점수 검증
└── 최종 수정

[PHASE 4: Output 저장]
└── 채널별 폴더로 정리하여 저장
```

## 📋 Channel-Specific Templates

### 1. 블로그 (Blog) - SEO 최적화
```markdown
# [메인 키워드] 포함 제목 (60자 이내)

## 메타 설명 (155자)
## 목차 (H2/H3 구조)
## 본문 (1,500-2,500자)
## CTA 섹션
## 관련 키워드: [키워드1], [키워드2]...
```

### 2. 인스타그램 카드뉴스
```markdown
# 카드뉴스 기획안

## 총 슬라이드: 7-10장
### 슬라이드 1: 후킹 (호기심 유발)
### 슬라이드 2-8: 핵심 내용
### 슬라이드 9: 요약
### 슬라이드 10: CTA + 저장 유도

## 캡션 (2,200자 이내)
## 해시태그 (20-30개)
```

### 3. 뉴스레터
```markdown
# 뉴스레터: [제목]

## 오프닝 훅 (3줄)
## 메인 콘텐츠
## 액션 아이템
## P.S. (추가 가치)
```

### 4. 트위터/X 스레드
```markdown
# 트위터 스레드 (15-20개 트윗)

🧵 1/ [강력한 오프닝]
2/ [맥락 설정]
...
15/ [핵심 요약]
마지막/ [CTA + 리트윗 요청]
```

### 5. 링크드인
```markdown
# 링크드인 포스트

## 후킹 문장 (첫 3줄이 중요)
## 스토리텔링 or 인사이트
## 불릿포인트 핵심
## CTA
## 해시태그 (3-5개)
```

## ⚙️ Execution Rules

1. **톤앤매너 일관성:** 모든 채널에서 브랜드 보이스 유지
2. **채널 특성 존중:** 플랫폼별 최적 형식 준수
3. **CTA 필수:** 모든 콘텐츠에 명확한 행동 유도 포함
4. **QA 필수:** 검증 없이 Output 저장 불가

- SEO Analysis (키워드 최적화)
- Image Prompt Generation (비주얼 기획)
- File Read/Write (문서 처리)
