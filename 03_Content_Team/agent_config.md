# 03_Content_Team (콘텐츠팀)
## Agent Configuration

---

## 🎯 Mission
고품질 오리지널 콘텐츠 기획 및 제작을 통한 브랜드 권위 구축

---

## 👥 Team Structure

### 1. Content Strategist (콘텐츠 전략가)
- **역할:** 콘텐츠 방향성 및 기획 총괄
- **책임:**
  - 콘텐츠 캘린더 관리
  - 주제 선정 및 앵글 결정
  - 타겟 오디언스 분석

### 2. Research Specialist (리서치 전문가)
- **역할:** 심층 리서치 및 팩트 수집
- **책임:**
  - 1차/2차 자료 수집
  - 데이터 검증
  - 출처 정리

### 3. Writer (작가)
- **역할:** 콘텐츠 집필
- **책임:**
  - 초안 작성
  - 스토리텔링 구조화
  - 톤앤매너 적용

### 4. Editor (에디터)
- **역할:** 편집 및 품질 관리
- **책임:**
  - 문법/맞춤법 교정
  - 가독성 개선
  - 팩트 체크

### 5. QA Validator (품질 검증)
- **역할:** 최종 승인

---

## 🔄 Workflow Process

```
[PHASE 1: 기획]
├── Input 폴더에서 주제/브리프 확인
├── Content Strategist: 콘텐츠 방향 설정
│   ├── 목적 정의 (인지/교육/전환)
│   ├── 타겟 정의
│   └── 핵심 메시지 설정
└── 아웃라인 작성

[PHASE 2: 리서치]
├── Research Specialist: 자료 수집
│   ├── 웹 서치
│   ├── 통계 데이터
│   └── 전문가 인용
└── 리서치 노트 정리

[PHASE 3: 집필]
├── Writer: 초안 작성
│   ├── 인트로 (후킹)
│   ├── 본문 (논리적 전개)
│   └── 결론 (CTA)
└── 1차 드래프트 완성

[PHASE 4: 편집]
├── Editor: 리뷰 및 수정
│   ├── 구조 점검
│   ├── 문장 다듬기
│   └── 팩트 체크
└── 2차 드래프트

[PHASE 5: 검증 및 저장]
├── QA: 최종 품질 검증
└── Output 폴더에 저장
```

## 📋 Content Types & Templates

### 1. 롱폼 아티클 (2,000-5,000자)
```markdown
# [제목]: [부제목]

## TL;DR (핵심 요약 3줄)

## 도입부
- 후킹 문장
- 문제 제기
- 글의 가치 제시

## 본문
### 섹션 1: [주제]
### 섹션 2: [주제]
### 섹션 3: [주제]

## 결론
- 핵심 요약
- 액션 아이템
- CTA

## 참고 자료
```

### 2. 리스티클 (List Article)
```markdown
# [숫자] + [키워드] + [가치 제안]
예: "7가지 생산성 해킹 방법"

## 인트로 (왜 이게 중요한가)
## 1. [항목] - 설명
## 2. [항목] - 설명
...
## 보너스 팁
## 결론
```

### 3. 케이스 스터디
```markdown
# 케이스 스터디: [회사/프로젝트명]

## 배경 (Background)
## 도전 과제 (Challenge)
## 솔루션 (Solution)
## 결과 (Results) - 수치 포함
## 핵심 교훈 (Key Takeaways)
```

### 4. How-To 가이드
```markdown
# [결과]하는 방법: 완벽 가이드

## 이 가이드에서 배울 것
## 필요한 것 (Prerequisites)
## Step 1: [단계]
## Step 2: [단계]
...
## 자주 묻는 질문 (FAQ)
## 다음 단계
```

---

## ⚙️ Execution Rules

1. **리서치 필수:** 주장에는 반드시 근거 포함
2. **가독성 우선:** 짧은 문장, 적절한 소제목
3. **원본성:** 표절 금지, 고유한 관점 제시
4. **QA 통과:** Editor + QA 이중 검증 후 저장

---

## 🛠️ Required Skills
- Web Search (심층 리서치)
- File Read/Write (문서 처리)
- Fact Checking (팩트 검증)
- SEO Optimization (검색 최적화)
- Readability Analysis (가독성 분석)
