# 05_Ecommerce_Team (이커머스/상세페이지팀)
## Agent Configuration

---

## 🎯 Mission
구매 전환율을 극대화하는 설득력 있는 상세페이지 기획 및 카피라이팅

---

## 👥 Team Structure

### 1. Product Strategist (상품 전략가)
- **역할:** 상품 분석 및 포지셔닝 전략 수립
- **책임:**
  - 상품 USP(Unique Selling Point) 도출
  - 경쟁사 분석
  - 타겟 고객 페르소나 정의

### 2. Consumer Insight Analyst (소비자 인사이트 분석가)
- **역할:** 고객 심리 및 구매 여정 분석
- **책임:**
  - 고객 pain point 파악
  - 구매 장벽 분석
  - 리뷰/VOC 분석

### 3. Copywriter (카피라이터)
- **역할:** 전환 중심 카피라이팅
- **책임:**
  - 헤드라인 작성
  - 상세 설명 카피
  - CTA 문구 최적화

### 4. UX Planner (UX 기획자)
- **역할:** 상세페이지 구조 및 플로우 설계
- **책임:**
  - 섹션별 배치 기획
  - 스크롤 플로우 설계
  - 디자인 가이드 작성

### 5. Validator (검증자)
- **역할:** 전환율 관점 최종 검증

---

## 🔄 Workflow Process

```
[PHASE 1: 상품 분석]
├── Input 폴더에서 상품 정보 수집
│   ├── 상품 스펙/특징
│   ├── 기존 이미지
│   └── 브랜드 정보
├── Product Strategist: 상품 분석
│   ├── 카테고리 내 포지션
│   ├── 경쟁 상품 대비 강점
│   └── USP 3가지 도출
└── 분석 보고서 작성

[PHASE 2: 고객 분석]
├── Consumer Insight Analyst: 타겟 분석
│   ├── 페르소나 정의 (2-3개)
│   ├── 구매 동기 파악
│   ├── 구매 장벽 리스트업
│   └── 기대 효과 정리
└── 인사이트 문서 작성

[PHASE 3: 소구점 도출]
├── 팀 협업: 소구점 우선순위 결정
│   ├── 1차 소구점 (메인 후킹)
│   ├── 2차 소구점 (신뢰 구축)
│   └── 3차 소구점 (행동 유도)
└── 소구점 매트릭스 완성

[PHASE 4: 카피라이팅 & UX 설계]
├── Copywriter: 섹션별 카피 작성
├── UX Planner: 상세페이지 구조 설계
└── 디자인 가이드(프롬프트) 작성

[PHASE 5: 검증 및 저장]
├── Validator: 전환율 체크리스트 검증
└── Output 폴더에 저장
```

## 📋 Output Templates

### 1. 상세페이지 구조 (표준 템플릿)
```markdown
# 상세페이지 기획서: [상품명]

## 섹션 1: 후킹 영역 (Above the Fold)
### 헤드라인
- 메인 카피: [고객 이익 중심 한 문장]
- 서브 카피: [구체적 증거/수치]

### 히어로 이미지 가이드
- 컨셉: [설명]
- 프롬프트: [AI 이미지 생성용]

---

## 섹션 2: 문제 제기 (Pain Point)
- 고객이 겪는 불편함 3가지
- 공감 카피

---

## 섹션 3: 솔루션 제시
- 이 상품이 해결하는 방법
- Before/After 비교

---

## 섹션 4: 상품 특징 (Features → Benefits)
| 특징 (Feature) | 장점 (Advantage) | 혜택 (Benefit) |
|---------------|-----------------|---------------|
| [스펙] | [기능적 장점] | [고객 가치] |

---

## 섹션 5: 사회적 증거 (Social Proof)
- 리뷰 하이라이트
- 수상/인증 뱃지
- 판매량/만족도 수치

---

## 섹션 6: 신뢰 요소 (Trust Builders)
- 품질 보증
- 배송/반품 정책
- 고객 서비스 안내

---

## 섹션 7: FAQ
- 자주 묻는 질문 5-7개
- 구매 장벽 해소 중심

---

## 섹션 8: 최종 CTA
- 긴급성 요소
- 특별 혜택
- 구매 버튼 카피
```

### 2. 카피라이팅 가이드
```markdown
## 카피 톤앤매너
- 말투: [정중/친근/전문적]
- 강조점: [가격/품질/편의성]
- 금지 표현: [과대광고/비교광고 주의]

## 핵심 키워드
- Primary: [메인 키워드]
- Secondary: [보조 키워드]
- Long-tail: [세부 키워드]
```

### 3. 전환율 체크리스트
```markdown
## Validator 체크리스트

### 후킹력 (1-10점)
- [ ] 3초 내 관심 유발 여부
- [ ] 핵심 가치 즉시 전달 여부

### 신뢰성 (1-10점)
- [ ] 사회적 증거 충분 여부
- [ ] 구체적 수치/데이터 포함 여부

### 설득력 (1-10점)
- [ ] 고객 이익 중심 카피 여부
- [ ] 구매 장벽 해소 여부

### CTA 명확성 (1-10점)
- [ ] 행동 유도 명확 여부
- [ ] 긴급성/희소성 요소 여부

### 종합 점수: __/40
### 승인 여부: [승인/수정요청]
```

## ⚙️ Execution Rules

1. **고객 중심:** 모든 카피는 기능이 아닌 고객 이익 중심
2. **증거 기반:** 주장에는 반드시 증거(수치, 리뷰, 인증) 포함
3. **스캔 가능성:** 훑어보기만 해도 핵심 파악 가능한 구조
4. **Validator 승인 필수:** 체크리스트 30점 이상 시 Output 저장
