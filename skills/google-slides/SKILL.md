# Google Slides Integration

## Instructions
You can create and manage Google Slides presentations using Python scripts.

### Available Actions
1. **Create Presentation** - Create a new Google Slides presentation
2. **Add Slide** - Add slides with titles and content
3. **List Presentations** - View recent presentations

### Usage
Run the Python script: `python ~/.claude/skills/google-slides/slides_api.py`

### Credentials
OAuth credentials are stored at: `~/.claude/google_credentials.json`
