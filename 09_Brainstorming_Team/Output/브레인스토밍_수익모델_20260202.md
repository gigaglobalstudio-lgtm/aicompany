## Panelist B 핵심 반박

> "아이디어는 좋지만 현실을 보죠.
>
> **문제 1: 시간**
> 1인 운영인데 컨설팅 하면서 강의 만들고 유튜브 하고
> 템플릿 만들 시간이 있나요?
>
> **문제 2: 집중**
> 10개 다 하면 10개 다 망합니다.
> 최대 2-3개에 집중해야 합니다.
>
> **문제 3: 검증**
> 고객이 정말 돈 내고 살까요?
> 무료로 풀리는 정보가 너무 많습니다."

## 공통적 문제점

1. **시간 부족:** 1인 운영의 근본적 한계
2. **집중 분산:** 다각화 ≠ 분산 투자
3. **검증 부재:** 수요 확인 없이 개발하면 실패
4. **경쟁 심화:** AI 관련 콘텐츠 레드오션

# Round 3: 변증법적 토론

## 논쟁 1: "지금 당장 뭘 해야 하나?"

| 발언자 | 주장 |
|--------|------|
| **A** | "템플릿 먼저. 이미 만들어진 10개 팀 시스템 포장만 하면 됨" |
| **B** | "템플릿 가격 하락 중. ₩10,000짜리가 ₩1,000 되는 시장" |
| **A** | "그래서 번들로 팔아야. 템플릿 + 강의 + 커뮤니티" |
| **B** | "번들은 좋은데 만들 시간이 없음. 컨설팅 하면서?" |
| **MC** | "컨설팅 줄이고 상품화에 시간 투자하는 건?" |
| **A** | "점진적 전환. 컨설팅 50% → 상품 50%로" |
| **B** | "수익 공백 기간이 위험. 백업 플랜 필요" |

**합의점:**
- 컨설팅 유지하면서 상품화 병행
- 첫 상품은 **이미 있는 것 포장** (시간 최소화)

## 논쟁 3: "패시브 인컴은 환상인가?"

| 발언자 | 주장 |
|--------|------|
| **B** | "패시브 인컴은 마케팅 용어. 유지보수 필요" |
| **A** | "100% 패시브는 아니어도 레버리지 효과는 있음" |
| **B** | "동의. 하지만 처음부터 패시브 노리면 안 됨" |
| **A** | "그래서 컨설팅 → 코칭 → 강의 → 템플릿 순서" |
| **B** | "순서 동의. 검증된 후에 상품화" |

**합의점:**
- 패시브 인컴은 **단계적으로** 구축
- 먼저 돈 받고 → 그 경험을 상품화

# 핵심 인사이트

### 전원 합의 사항

1. **컨설팅 유지 필수** - 당장의 수익원
2. **최대 2-3개 집중** - 분산은 실패
3. **검증 후 상품화** - 고객 반응 먼저
4. **단계적 전환** - 급하게 하면 망함

## 선정된 수익 모델 (Top 3)

### 1순위: 그룹 코칭 프로그램 ⭐

**이름:** "AI 에이전트 부트캠프" (4주 과정)

| 항목 | 내용 |
|------|------|
| 형태 | 주 1회 라이브 (2시간) + 커뮤니티 |
| 정원 | 10-15명 |
| 가격 | ₩300,000/인 |
| 예상 매출 | ₩3-4.5M/기수 |
| 투자 시간 | 주 4시간 (라이브 2h + 준비 2h) |
| 스케일업 | 월 2기수 → 월 ₩6-9M |

**Why 1순위?**
- 컨설팅 경험 즉시 활용 가능
- 1:1보다 효율적
- VOD 전환 가능 (녹화)
- 커뮤니티 → 멤버십 전환

**리스크:** 첫 기수 모객 어려움
**대응:** 얼리버드 50% 할인, 컨설팅 고객 초대

---

### 2순위: 템플릿 번들

**이름:** "My AI Company 스타터 키트"

| 항목 | 내용 |
|------|------|
| 구성 | 10개 팀 템플릿 + 사용법 가이드 + 영상 3개 |
| 가격 | ₩49,000 (런칭 할인 ₩29,000) |
| 판매처 | 자사몰 (Gumroad/Lemon Squeezy) |
| 예상 판매 | 월 50개 → ₩1.5-2.5M |
| 투자 시간 | 초기 20시간 (포장) + 월 2시간 (업데이트) |

**Why 2순위?**
- 이미 만들어진 것 활용
- 한 번 만들고 계속 판매
- 코칭 참가자에게 번들 제공 가능

**리스크:** 무료 템플릿과 경쟁
**대응:** 실제 사용 영상 + 커뮤니티 접근권 추가

### 3순위: 뉴스레터 + 어필리에이트

**이름:** "AI 에이전트 위클리"

| 항목 | 내용 |

**리스크:** 성장에 시간 소요
**대응:** 코칭/템플릿 구매자 자동 등록
