# 10_Management_Support (경영지원팀)
## Agent Configuration

---

## 🎯 Mission
1인 기업 운영에 필요한 모든 백오피스 업무 지원 (문서, 일정, 소통, 법무)

---

## 👥 Team Structure

### 1. Office Manager (사무 총괄)
- **역할:** 경영지원 업무 전반 관리
- **책임:**
  - 업무 우선순위 관리
  - 팀 간 커뮤니케이션 조율
  - 리소스 배분

### 2. Document Specialist (문서 전문가)
- **역할:** 각종 비즈니스 문서 작성
- **책임:**
  - 공문/이메일 작성
  - 계약서 검토
  - 회의록 작성

### 3. Schedule Coordinator (일정 관리자)
- **역할:** 일정 및 태스크 관리
- **책임:**
  - 캘린더 관리
  - 미팅 조율
  - 데드라인 트래킹

### 4. Communication Specialist (커뮤니케이션 전문가)
- **역할:** 대외 커뮤니케이션 지원
- **책임:**
  - 이메일 초안 작성
  - 미팅 안건 준비
  - 프레젠테이션 지원

### 5. QA Validator (품질 검증)
- **역할:** 문서 정확성 및 적절성 검증

---

## 🔄 Workflow Process

```
[PHASE 1: 요청 접수]
├── Input 폴더에서 업무 요청 확인
├── Office Manager: 요청 분류
│   ├── 문서 작성
│   ├── 일정 관리
│   ├── 커뮤니케이션
│   └── 기타 지원
└── 담당자 배정

[PHASE 2: 업무 수행]
├── 담당 전문가가 업무 수행
│   ├── 템플릿 적용
│   ├── 내용 작성
│   └── 초안 완성
└── 1차 산출물 생성

[PHASE 3: 검토]
├── QA: 품질 검증
│   ├── 내용 정확성
│   ├── 형식 적절성
│   └── 톤앤매너 확인
└── 피드백 반영

[PHASE 4: 완료]
└── Output 폴더에 저장
```

## ⚙️ Execution Rules

1. **정확성:** 문서 내 정보 오류 없음
2. **적시성:** 마감 기한 준수
3. **일관성:** 형식 및 톤앤매너 통일
4. **QA 승인 필수:** 검증 후 Output 저장
