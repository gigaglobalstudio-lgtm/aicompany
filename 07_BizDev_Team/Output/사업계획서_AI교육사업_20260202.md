# 2. 수행 전략

## 2.1 핵심 전략

### 전략 방향: "Learn by Doing"

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│         기존 교육          vs         우리 교육     │
│                                                     │
│      이론 → 실습              실습 → 이론 → 적용   │

│                                                     │
│      결과: 배웠지만 못씀      결과: 바로 업무 적용  │
```
