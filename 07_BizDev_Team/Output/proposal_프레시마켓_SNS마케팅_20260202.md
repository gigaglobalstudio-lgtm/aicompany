# 1. Executive Summary

## 핵심 제안

> **"콘텐츠가 매출이 되는 SNS 마케팅"**
>
> 프레시마켓의 프리미엄 브랜드 가치를 살리는 콘텐츠 전략과
> 데이터 기반 퍼포먼스 마케팅의 결합으로
> 6개월 내 SNS 경유 매출 비중 30%를 달성하겠습니다.

---

## 제안 개요

| 항목 | 내용 |
|------|------|
| **서비스 범위** | 인스타그램 + 네이버 블로그 + 틱톡(신규) 통합 관리 |
| **계약 기간** | 6개월 (2026.03 ~ 2026.08) |
| **월 비용** | 450만 원 (VAT 별도) |
| **핵심 성과 목표** | 인스타 3만 팔로워 / 블로그 일 2,000명 / 매출 비중 30% |

---

## 왜 저희인가?

1. **식품 이커머스 전문 경험** - 유사 업종 10개 이상 수행 실적
2. **콘텐츠 + 퍼포먼스 통합** - 브랜딩과 매출 두 마리 토끼
3. **데이터 기반 운영** - 주간 리포트 + 월간 전략 리뷰

# 2. 귀사 현황 분석

## 2.1 SWOT 분석

### Strengths (강점)
- 프리미엄 식품 포지셔닝 (차별화된 상품력)
- 충성 고객층 존재 (재구매율 추정 40% 이상)
- 자사몰 + 스마트스토어 듀얼 채널 운영

### Weaknesses (약점)
- SNS 콘텐츠 퀄리티/일관성 부족
- 브랜드 스토리텔링 미흡
- 채널별 최적화 전략 부재
- 성과 측정 체계 미비

### Opportunities (기회)
- 건강식품 시장 연 15% 성장세
- MZ세대 건강 관심 증가
- 숏폼 콘텐츠를 통한 바이럴 가능성
- 틱톡 푸드 카테고리 성장

### Threats (위협)
- 대형 플랫폼(마켓컬리, 쿠팡) 경쟁 심화
- 콘텐츠 제작 비용 상승
- 플랫폼 알고리즘 변화

## 3.3 콘텐츠 전략

### 콘텐츠 믹스 (Content Mix)

```
┌────────────────────────────────────────────────┐
│                                                │
│   40% │████████████│ 레시피/활용법 콘텐츠     │
│   25% │███████     │ 상품 스토리/소개         │
│   20% │██████      │ 생산자/Behind 스토리     │
│   10% │███         │ 이벤트/프로모션          │
│    5% │██          │ 고객 후기/UGC            │
│                                                │
└────────────────────────────────────────────────┘
```

### 콘텐츠 유형별 상세

**1. 레시피/활용법 콘텐츠 (40%)**
- 계절 과일 활용 디저트 레시피
- 10분 건강 한 끼 (유기농 채소 활용)
- 건강식품 활용 가이드

**2. 상품 스토리 (25%)**
- 신상품 런칭 콘텐츠
- 베스트셀러 재조명
- 제철 식재료 큐레이션

**3. 생산자/Behind 스토리 (20%)**
- 농장 방문기
- 품질 관리 과정
- 직원 추천 상품

**4. 이벤트/프로모션 (10%)**
- 시즌 프로모션
- 팔로워 이벤트
- 라이브 커머스 예고

**5. 고객 후기/UGC (5%)**
- 고객 요리 사진 리그램
- 리뷰 하이라이트
- 언박싱 영상
